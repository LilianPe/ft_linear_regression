# ft_linear_regression

## Launch it

```
bash
make
```
or 
```
bash
make re
```
to init the values that estimatePrice.py use

Then launch

```
bash
python3 ft_linear_regression.py
```

to train the model

finally, you can do

```
bash
python3 estimatePrice.py
```
Then enter a mileage and it will give you the predicted price for it!
